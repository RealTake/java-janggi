# 요구 사항

- [x] x와 y 좌표를 가지고 있는 점을 생성한다.
    - [x] x 좌표는 0부터 8까지의 범위를 가진다.
    - [x] y 좌표는 0부터 9까지의 범위를 가진다.
    - [x] 범위를 벗어난 점을 생성할 경우 예외 처리한다.
    - [x] x와 y의 정해진 범위에 맞게 점을 생성 해둔다.


- [x] 기물은 존재 하는 좌표 내에서 움직일 수 있다.
- [x] 기물은 기본 적으로 다른 기물을 뛰어 넘을 수 없다.
- [x] 기물은 같은 나라의 기물이 있는 위치로 이동할 수 없다.
- [x] 기물이 다른 나라의 기물이 있는 위치로 이동 한다면 해당 기물을 잡을 수 있다.


- [x] 기물은 차(Chariot), 상(Elephant), 마(Horse), 사(Advisor), 장(King), 포(Cannon), 병(Pawn)이 존재한다.
- [x] 기물마다 다른 이동 전략을 가진다.
    - [x] 차는 칸수 제한 없이 직선으로만 이동한다.
    - [x] 상은 상/하/좌/우로 한칸, 한쪽 대각선 방향으로 2칸 이동 한다.
    - [x] 마는 상/하/좌/우로 한칸, 한쪽 대각선 방향으로 1칸 이동 한다.
    - [x] 사는 상/하/좌/우로 한칸 이동 한다.
    - [x] 장은 상/하/좌/우로 한칸 이동 한다.
    - [x] 포는 칸수 제한 없이 직선으로만 이동한다.
        - [x] 이동 경로에 말이 1개 존재해야 한다.
        - [x] 포끼리는 잡을 수도 없고 뛰어 넘을 수도 없다.
    - [x] 병은 상대방을 바라보는 기준에서 상/좌/우 1칸씩만 이동할 수 있다.


- [x] 각각의 기물은 나라를 가진다.
    - [x] 나라는 한나라(Han)와 초나라(Cho)로 나뉜다.


- [x] 초기 기물을 아래와 같이 배치 한다.
    - [x] 한나라의 말을 배치 한다.
    - [x] 초나라의 말을 배치 한다.

```배치 예시

8  ＿ ＿ ＿ ＿ 장 ＿ ＿ ＿ ＿

2  ＿ 포 ＿ ＿ ＿ ＿ ＿ 포 ＿
1  ＿ ＿ ＿ ＿ 장 ＿ ＿ ＿ ＿
0  차 상 마 사 ＿ 사 마 상 차
   ０ １ ２ ３ ４ ５ ６ ７ 8
```

- [x] 게임 진행 순서는 초나라부터 시작한다.
- [x] 장(King)이 잡힌다면 잡은 쪽이 승리하며 게임이 종료된다.
- [x] 한 턴 스킵할 수 있다. (말을 움직이지 않고 머무를 수 있다.)
    - [x] 2번 연속 턴이 스킵된다면 무승부로 게임이 종료된다.
- [x] 게임 진행 중에 한 턴 무를 수 있다
    - [x] 무를 수 있는 턴이 없다면 예외를 발생시킨다
- [x] 게임 진행 중에 각 나라의 현재 점수를 구할 수 있다
- [x] 게임이 끝난다면 게임 결과를 구할 수 있다
    - [x] 게임 결과는 한나라 승, 초나라 승, 무승부로 나뉜다


- [x] 궁성이라는 공간이 존재한다
    - [x] 초나라의 궁성은 다음과 같다
      ```
      (3, 2), (4, 2), (5, 2)
      (3, 1), (4, 1), (5, 1)
      (3, 0), (4, 0), (5, 0)
      ```
    - [x] 한나라의 궁성은 다음과 같다
      ```
      (3, 9), (4, 9), (5, 9)
      (3, 8), (4, 8), (5, 8)
      (3, 7), (4, 7), (5, 7)
      ```
- [x] 각 궁성의 중심((4, 1), (4, 8))으로부터의 한칸 대각선 이동은 가로 세로로만 이동 가능한 기물도 이동 가능하다
- [x] 각 궁성의 중심((4, 1), (4, 8))으로 향하는 한칸 대각선 이동은 가로 세로로만 이동 가능한 기물도 이동 가능하다
- [x] 각 궁성의 중심((4, 1), (4, 8))을 통하는 두칸 대각선 이동은 가로 세로로만 이동 가능한 기물도 이동 가능하다
- [x] 마, 상에게 궁성에 대한 규칙은 적용되지 않는다. (대각 이동을 가로 세로 이동으로 간주할 수 없다)
- [x] 장, 사는 궁성을 벗어날 수 없다
- [x] 병은 궁 안의 대각 이동이더라도 뒤로 이동이 불가능하다

- [x] 게임 시작 시 저장된 게임과 새 게임 시작을 선택할 수 있다.
- [x] 저장된 게임을 불러올 경우 마지막 상태의 장기판으로부터 게임을 진행한다
