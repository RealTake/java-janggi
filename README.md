# java-janggi

장기 미션 저장소

## 기능 구현 목록

### 보드

- [x] 특정 위치에서 이동 가능한 위치 리스트를 반환한다
- [x] 점수를 계산 한다
- [x] 예외 처리
  - [x] 해당 위치에 기물이 없으면 예외

### 기물들의 위치

- [x] 기물들의 위치를 초기화 한다
  - [x] 게임 진행 이력이 있는 경우, 이어서 진행한다.
  - [x] 게임 진행 이력이 없는 경우, 처음부터 진행한다.
- [x] 게임이 종료되면 모든 기물 위치를 삭제한다.

### 기물

- [x] 이동 가능한 위치를 리스트로 반환

  - [x] 포 (`Cannon`)
    - [x] 포는 다른 장애물이 존재하지 않는 경우 움직일 수 없다.
    - [x] 포는 다른 장애물이 있는 경우 이를 뛰어넘을 수 있다.
    - [x] 포는 궁성 내부에서 대각선으로도 이동할 수 있다.
  - [x] 차 (`Chariot`)
    - [x] 차는 이동 방향의 끝까지 움직일 수 있다.
    - [x] 차는 장애물을 만날 때까지 움직일 수 있다.
    - [x] 차는 궁성 내부에서 대각선으로도 이동할 수 있다. 이때, 궁성 외부로는 나갈 수 없다.
  - [x] 상 (`Elephant`)
    - [x] 상은 다른 장애물을 뛰어넘을 수 없다.
    - [x] 상은 궁성 길과 무관하게 움직일 수 있다.
  - [x] 졸/병 (`Guard`)
    - [x] 한나라 기물은 좌/우/아래 방향으로 움직일 수 있다.
    - [x] 초나라 기물은 좌/우/위 방향으로 움직일 수 있다.
    - [x] 궁성 중심, 꼭짓점 위치에서 대각선 방향으로도 움직일 수 있다.
  - [x] 마 (`Horse`)
    - [x] 마는 다른 장애물을 뛰어넘을 수 없다.
    - [x] 마는 궁성 길과 무관하게 움직일 수 있다.

### 데이터베이스

- [x] 데이터베이스 작업이 실패하더라도 게임을 진행할 수 있다.
