# java-janggi

## 프로그램 실행 흐름

### 프로그램 시작

- 프로그램 시작 안내 메시지를 출력한다.

  ```
  장기 게임에 오신 것을 환영합니다.
  ```
  
### 저장된 게임 진행 여부 입력
- 저장된 게임이 있다면 진행 여부를 입력 받는다.

  ```
  > 저장된 게임이 있습니다. 다시 시작하시겠습니까? (예: y/n)
  > n 입력 시 저장된 게임은 사라지고, 새 게임을 시작합니다.
  y
  ```

### 게임 진행 커맨드 입력
- 게임 진행 커맨드를 입력 받는다.

  ```
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  move
  ```
  
- 예외
  - 올바르지 않은 입력이 들어온 경우 에러 메시지를 출력한다.
  
  ```
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  gohome
  
  [gohome]: [ERROR] 올바른 진행 커맨드를 입력해주세요.
  ```

### 상마 순서 입력

- 상마 순서 결정을 위한 입력을 받는다.
- 새 게임 시작 시에만 입력 받는다.

  ```
  > 한나라 상마 순서 번호를 입력해주세요. (예: 3)
  1. 상마상마
  2. 상마마상
  3. 마상상마
  4. 마상마상
  1
  
  > 초나라 상마 순서 번호를 입력해주세요. (예: 3)
  1. 상마상마
  2. 상마마상
  3. 마상상마
  4. 마상마상
  2
  ```

- 예외
    - 올바르지 않은 입력이 들어온 경우 에러 메시지를 출력한다.

  ```
  > 초나라 상마 순서 번호를 입력해주세요. (예: 3)
  1. 상마상마
  2. 상마마상
  3. 마상상마
  4. 마상마상
  6
  [6]: [ERROR] 올바른 상마 순서 번호를 입력해주세요.
  ```

### 기물 위치 초기화

- 입력한 상마 순서에 따라 기물 위치를 초기화한다.

### 이동 커맨드 입력

- 기물 이동 시(move 진행 커맨드 입력 시)에만 입력 받는다.
- src dst 양식으로 입력해야 한다.
- src에 있는 내 기물을 dst로 이동한다.

  ```
  > 현재 턴 : 초나라
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  move
   
  > 이동할 기물의 현재 위치와 이동할 위치를 입력해주세요. (예: 1,1 2,1)
  1,1 2,1
  ```

- 예외
    - src가 빈 칸인 경우 예외가 발생한다.
    - src에 상대 기물이 있는 경우 예외가 발생한다.
    - src에서 dst로 이동할 수 없는 경우 예외가 발생한다.

### 보드 출력

- 현재 장기판 상태를 출력한다.
- 팀은 색깔로 구분한다.
    - 초나라 : 초록색
    - 한나라 : 빨간색

  ```
  차 상 마 사 ㅁ 사 상 마 차
  ㅁ ㅁ ㅁ ㅁ 왕 ㅁ ㅁ ㅁ ㅁ
  ㅁ 포 ㅁ ㅁ ㅁ ㅁ ㅁ 포 ㅁ
  병 ㅁ 병 ㅁ 병 ㅁ 병 ㅁ 병
  ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ
  ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ
  병 ㅁ 병 ㅁ 병 ㅁ 병 ㅁ 병
  ㅁ 포 ㅁ ㅁ ㅁ ㅁ ㅁ 포 ㅁ
  ㅁ ㅁ ㅁ ㅁ 왕 ㅁ ㅁ ㅁ ㅁ
  차 상 마 사 ㅁ 사 마 상 차
  ```

### 점수 출력

- 사용자가 점수 출력을 요청하면(status 진행 커맨드 입력 시) 팀별 장기의 점수를 계산하여 출력한다.
- 승패와 관계 없이 출력한다.
- | 기물 |차|포|마|상|사|병|후수(한나라)|
  |:--:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
  | **점수** |13|7|5|3|3|2|1.5|

  ```
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  status
  
  한나라 : 71.5점
  초나라 : 70.0점
  ```
  
### 게임 저장

- 사용자가 게임 저장을 요청하면(save 진행 커맨드 입력 시) 현재 게임을 저장한다.

  ```
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  save
  
  장기 게임을 저장 후 종료합니다.
  ```

### 승패 결과 출력

- 이동 중간에 왕이 죽으면 승패 결과를 출력한다.

  ```
  초나라의 승리입니다.
  ```
  
### 게임 종료 출력

- 게임 종료 시(exit 진행 커맨드 입력 시) 승패 결과와 함께 안내 메시지를 출력한다.

  ```
  > 원하는 진행 커맨드를 입력해주세요. (예: move)
  - 기물 이동: move
  - 점수 확인: status
  - 저장 후 종료: save
  - 저장하지 않고 게임 종료: exit
  exit
  
  한나라 : 71.5점
  초나라 : 70.0점
  
  한나라의 승리입니다.
  장기 게임을 종료합니다.
  ```
