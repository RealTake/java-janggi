# java-janggi

## 실행 환경 구축

./docker/docker-compose.yml가 존재하는 경로에서 아래의 명령어를 먼저 입력해야 데이터베이스 관련 로직이 동작합니다.

docker 컨테이너 실행

```shell
docker-compose -p janggi up -d
```

docker 컨테이너를 종료

```shell
docker-compose -p janggi down
```

## 프로그램 실행 흐름

### 방 이름 입력

- 접속하려는 방 이름을 입력받는다.

```
입장할 게임방 이름을 입력해주세요.
> abc

'abc'방을 새로 생성하고 있습니다. 
게임을 이어서 하려면 다음에 접속할 때 동일한 방 이름을 입력해 접속해 주세요.
```

### 상마 순서 입력

- 상마 순서 결정을 위한 입력을 받는다.

```
초나라 상마 순서를 입력해주세요. (예: 3)
1. 상마상마
2. 상마마상
3. 마상상마
4. 마상마상
> 2

한나라 상마 순서를 입력해주세요. (예: 3)
1. 상마상마
2. 상마마상
3. 마상상마
4. 마상마상
> 4
```

- 예외
    - 올바르지 않은 입력이 들어온 경우 에러 메시지 출력 후 재입력을 받는다.

```
초나라 상마 순서를 입력해주세요. (예: 3)
1. 상마상마
2. 상마마상
3. 마상상마
4. 마상마상
6
[ERROR] 올바른 상마 순서 번호를 입력해주세요. (예: 3)
```

### 게임 시작

- 게임 시작 안내 메시지를 출력한다.

```
장기 게임에 오신 것을 환영합니다.
방 'abc'에 연결되었습니다.
```

### 기물 위치 초기화

- 입력한 상마 순서에 따라 기물 위치를 초기화한다.

### 메인 화면에서 원하는 기능 선택

- `move`, `status`, `end`를 입력 받을 수 있다.
- 입력 받은 커맨드에 맞는 기능을 수행한다.

```
원하는 기능을 선택해주세요. (예: move)
> move: 기물 이동
> status: 현재 점수 출력
> end: 게임 종료
```

### 기물 이동 (move)

- `src dst` 형식으로 입력해야 한다.
- src에 있는 내 기물을 dst로 이동한다.

```
현재 턴: 초나라
이동할 기물의 현재 위치와 이동할 위치를 입력해주세요. (예: 7,1 7,2)
7,1 7,2
```

- 예외
    - 좌표를 잘못된 형식으로 입력하면 에러 메시지 출력 후 재입력을 받는다.
    - src 좌표에 아무 기물이 없는 경우 에러 메시지 출력 후 재입력을 받는다.
    - src에 상대 기물이 있는 경우 에러 메시지 출력 후 재입력을 받는다.
    - src에서 dst로 이동할 수 없는 경우 에러 메시지 출력 후 재입력을 받는다.

- 이동 중간에 왕이 죽으면 승패 결과를 출력한다.

```
초나라의 승리입니다.
```

### 현재 점수 출력(status)

- 현재 장기판의 상태를 기준으로 각 나라의 점수를 출력한다.

```
초나라: 10.0점
한나라: 12.5점
```

### 게임 종료(end)

- 살아남은 기물들의 점수 합으로 승패를 결정한다.
- 점수 합이 더 높은 사람이 승리한다.
- 각 기물의 점수
    * 차: 13점
    * 포: 7점
    * 마: 5점
    * 상: 3점
    * 사: 3점
    * 병: 2점

```
초나라: 10.0점
한나라: 12.5점

한나라의 승리입니다.
```
