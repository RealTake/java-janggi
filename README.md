# java-janggi

장기 미션 저장소

- [x] 데이터베이스에서 모든 장기들을 조회한다.
- [x] 데이터베이스에서 선택한 장기 게임에 해당하는 장기를 찾는다.

### Janggi
- [x] 현재 턴에 해당하는 팀의 기물 이동을 진행하고 턴을 바꾼다.
  - [x] 데이터베이스에 해당 장기 게임턴을 저장한다.
- [x] 원하는 팀의 점수를 알려준다.
- [x] 게임이 끝난는지 판단한다.
- [x] 승리한 팀을 알려주고, 게임이 끝나지 않았다면 예외를 발생시킨다.

### Board
- [x] 장기판 초기화
  - 장기 게임을 처음 생성한 경우 
    - [x] 장기판에 초기 배치 상태에 맞게 모든 기물을 생성한다.
    - [x] 데이터베이스에 초기 배치된 모든 기물의 정보를 저장한다.
  - 진행중이던 장기 게임을 불러온 경우
    - [x] 데이터베이스에 저장된 모든 기물의 정보를 불러온다.
- [x] 기물을 이동시킨다.
  - [x] 선택한 위치와 이동할 위치가 동일하다면 예외를 발생시킨다.
  - [x] 선택한 위치에 기물이 없다면 예외를 발생시킨다.
  - [x] 선택한 위치의 기물이 다른 팀이라면 예외를 발생시킨다.
  - [x] 이동시킬 위치의 기물이 같은 팀의 기물이라면 예외를 발생시킨다.
  - [x] 기물의 이동 경로에 허용되지 않은 장애물이 있다면 예외를 발생시킨다.
  - [x] 잡는 기물이 잡을 수 없는 기물이라면 예외를 발생시킨다.
  - [x] 이동하려는 위치에 적 기물이 있다면 해당 기물을 삭제하고, 선택한 위치의 기물을 삭제한 후, 선택한 위치의 기물을 이동하려는 위치로 이동시킨다.
    - [x] 이동하려는 위치에 적 기물이 있다면 해당 기물을 삭제한다.
    - [x] 선택한 위치의 기물을 이동시킬 위치로 이동시킨다.
    - [x] 데이터베이스의 이동 위치 기물/선택 위치 기물을 삭제하고, 이동시키려는 위치에 기물을 저장한다.
- [x] 현재 팀 별 점수를 계산한다.
- [x] 현재 살아있는 왕들을 찾는다.

### Cannon
- [x] 보드판의 선을 따라 이동한 것이 아니라면 예외를 발생시킨다.
- [x] 이동 경로상의 장애물이 1개여야만 이동할 수 있다.
- [x] 이동 경로상의 장애물이 포라면 이동할 수 없다.
- [x] 잡으려는 기물이 포라면 잡을 수 없다.

### Chariot
- [x] 보드판의 선을 따라 이동한 것이 아니라면 예외를 발생시킨다.

- [x] 이동 경로상에 장애물이 없어야 이동할 수 있다.
