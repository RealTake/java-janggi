# Java-Janggi

## 장기 미션 저장소

## 실행 방법

1. Docker를 실행한다.
2. 터미널에 아래의 명령어를 입력한다.
    ```bash
      docker-compose up -d
    ```
3. JanggiApplication을 실행한다.


## 장기 기획서

- 궁(宮) 또는 궁성(宮城)의 영역을 고려한다.
    - 궁 기물과 사 기물은 궁성 내에서만 이동할 수 있다.
    - 궁성 내 꼭짓점과 중앙에서는 대각선 이동이 가능하다.
- 게임 승리 조건
    - 궁 기물을 먼저 잡으면 점수에 상관없이 승리한다.
    - 사용자가 게임 종료를 입력하면 남은 기물의 점수로 승패를 판단한다.
- 데이터베이스를 연결한다.
    - 사용자가 게임 저장을 입력하면 게임 정보가 데이터베이스에 저장된다.
    - 진행 중인 게임이 있다면 해당 게임을 자동으로 불러온다.

## 기능 구현

### 장기

- [x] 왕이 잡혔거나 게임 종료를 입력하면 게임을 종료한다.
    - [x] 왕을 잡았다면 잡은 쪽이 승리한다.
    - [x] 게임 종료를 입력하면 남은 기물로 승패를 판단한다.
        - [x] 남은 기물의 점수 총합으로 점수를 계산한다.
        - [x] 후수(한나라)의 경우 기물 당 1.5점이 추가로 부여된다.

### 기물들

- [x] 기물을 초기화한다.
- [x] 어떤 좌표에 기물이 존재하는지 확인한다.

### 기물 (Unit)

- [x] 기물이 갈 수 있는 경로를 모두 계산한다.
    - [x] **궁**: 궁성 내 1번. (외부 이동 불가)
    - [x] **사**: 궁성 내 1번. (외부 이동 불가)
    - [x] **졸**: 직진(좌, 우, 상) 1번.
    - [x] **마**: 직진(좌, 우, 상, 하) 1번 후 대각선 이동 1번.
    - [x] **차**: 직진(좌, 우, 상, 하)만 가능하다.
    - [x] **포**: 무조건 기물을 한 칸 점프해야 하며, 직진만 가능하다.
    - [x] **상**: 직진(좌, 우, 상, 하) 1번 후 대각선 이동 2번.

### 기물 타입

### 궁성

- [x] 궁성의 네 꼭짓점과 중앙에서는 궁성 내 대각선으로 이동 가능하다.

``` plaintext
한나라 궁성
[(3,0),(4,0),(5,0)]
[(3,1),(4,1),(5,1)]
[(3,2),(4,2),(5,2)]

초나라 궁성
[(3,7),(4,7),(5,7)]
[(3,8),(4,8),(5,8)]
[(3,9),(4,9),(5,9)]
```

### 좌표

- [x] 좌표가 보드를 넘는지 판단한다.

### 게임 관리

- [x] 종료되지 않은 게임이 있다면 자동으로 불러온다.
- [x] 게임을 저장하면 DB에 게임 정보가 반영된다.

## 참고

### 기물 별 대표 문자

``` plaintext
궁 (將, 帥)  : K
졸, 병 (卒, 兵) : A
사 (士, 仕)  : S
마 (馬)  : H
차 (車)  : C
포 (包)  : B
상 (象)  : E
```

### 기물 별 점수

- 승패와 별개로 기물 별 점수를 계산한다.
- 후수(한나라)는 아래 점수표에서 1.5 더해 계산한다.

```plaintext
차: 13점
포: 7점
마: 5점
상, 사: 3점
병: 2점
```
