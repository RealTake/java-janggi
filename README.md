# java-janggi

장기 미션 저장소

### 초기화

- [x] 장기판에 말들에 대한 초기 배치 상태를 결정한다.
- [x] 장기판에 초기 배치 상태에 맞게 모든 말을 생성한다.
- [x] 게임을 저장한다.

### 이동

- [x] 이동하려는 기물이 턴에 맞는 팀인지 확인한다.
- [x] 좌표를 받아 이동할 수 있는지 확인한다.
- [x] 보드에서 기물의 위치를 바꾼다.
- [x] 기물을 움직이고 나면 다음 팀의 턴으로 변경한다.
- [x] 게임을 저장한다.

### 점수

- [x] 팀 별로 점수를 계산한다.

### 게임 종료

- [x] 상대 팀의 왕을 잡으면 게임이 종료된다.
- [x] 게임 종료시, 저장된 게임을 삭제한다.

### 입출력

- [x] 팀 별로 점수를 출력한다.
- [x] 현재 턴을 출력한다.
- [x] 장기판을 출력한다.
- [x] 사용자로부터 좌표를 입력받고 이동한다.
- [x] 게임 종료 시 승리한 팀을 출력한다.
