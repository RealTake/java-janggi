# java-janggi

장기 미션 저장소

## DB 실행 방법

도커: docker-compose -p janggi up -d
어플리케이션 실행

## 기능 요구사항

### 보드판

- 크기: 9 * 10
- 초나라 한나라 모두 마상상마로 시작

### 점수

- 전차 13점
- 대포 7점
- 말 5점
- 상 3점
- 사 3점
- 병 2점

### 플레이어 턴

- 플레이어의 기물을 움직이면 턴이 바뀐다.
- 상대방의 장을 잡으면 게임이 종료된다.

### 기물

- 마
    - 한 칸 전진 후 대각선으로 움직인다.
    - 진행 경로에 기물이 있으면 움직이지 못한다.
- 상
    - 한 칸 전진 후 두 칸 대각선으로 움직인다.
    - 진행 경로에 기물이 있으면 움직이지 못한다.
- 차
    - 수직, 수평으로 움직인다.
    - 기물을 뛰어넘을 수 없다.
- 포
    - 기물 하나만 뛰어넘을 수 있다.
    - 포는 뛰어넘을 수 없고, 포를 잡을 수 없다.
- 병
    - 뒤를 제외하고 앞 옆 한 칸 움직인다.
- 장군
    - 자신으로 부터 한 칸 움직인다.
- 사
    - 자신으로 부터 한 칸 움직인다.

## 입력

- 움직이려는 기물의 좌표를 입력한다. ex) 1행 1열 -> 11
- 기물을 선택 후 이동하려는 좌표를 입력한다. ex) 3행 5열 -> 35
- 10행은 0으로 입력한다.
- 올바르지 않은 좌표를 입력한 경우 처음부터 다시 입력 받는다.

## 출력

- 가로선은 위에서부터 아래로 1부터 0까지(1,2,3..,9,0) 표기
- 세로줄은 각각 왼쪽에서 오른쪽으로 1부터 9까지(1,2,3..,9) 표기
- 기물이 존재하는 위치에 기물의 이름을 한글으로 출력
- 각 턴마다 점수를 출력한다.

## 게임 저장

- -1을 입력하여 정상적으로 게임을 종료한 경우 자동으로 게임이 저장된다
- 게임을 시작 시 저장되어 있는 게임을 불러오거나, 생성할 수 있다.
- 게임이 종료되면 저장된 게임은 삭제된다.
- 한 페이지에 최대 5개 게임만 불러온다.
