# Java-Janggi

## 장기 미션 저장소

### 과제 진행 요구 사항

첫 시작 시 장기의 보드판과 각 기물을 구현하라.

#### 장기 기획서

- 1단계에서는 장기 기물과 보드 초기화를 구현한다.
- 개발을 시작하기 전에 어떤 기능을 구현해야 하는지 정리한다.
- 궁(宮) 또는 궁성(宮城)의 영역은 구현하지 않는다.
    - 즉, 장기판의 왕(將, 帥)과 사(士, 仕)의 이동과 관련된 규칙은 고려하지 않는다.

---

## 기능 구현

### 기물들 (Janggi)

- [x] 기물을 초기화한다.
- [x] 어떤 좌표에 기물이 존재하는지 확인한다.

### 기물 (Unit)

- [x] 기물이 갈 수 있는 경로를 모두 계산한다.
    - [x] **졸**: 직진(좌, 우, 상) 1번.
    - [x] **마**: 직진(좌, 우, 상, 하) 1번 후 대각선 이동 1번.
    - [x] **차**: 직진(좌, 우, 상, 하)만 가능하다.
    - [x] **포**: 무조건 기물을 한 칸 점프해야 하며, 직진만 가능하다.
    - [x] **상**: 직진(좌, 우, 상, 하) 1번 후 대각선 이동 2번.

### 좌표 (Position)

- [x] 좌표가 보드를 넘는지 판단한다.

## 참고

### 기물 별 대표 문자

``` plaintext
궁 (將, 帥)  : K
졸, 병 (卒, 兵) : A
사 (士, 仕)  : S
마 (馬)  : H
차 (車)  : C
포 (包)  : B
상 (象)  : E
```

### 그 외

- 어느 팀의 기물인지는 문자의 색상으로 구분한다
    - 빨간색: 한나라
    - 파란색: 초나라
- 궁(將, 帥)과 사(士, 仕)는 구현하지 않으므로 논외(N)로 둔다
