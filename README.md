# java-janggi

장기 미션 저장소

### 장기 게임

- [x] 홍과 청이 번갈아가면서 게임을 진행한다.
    - [x] 청이 먼저 시작한다.
- [x] 상대방의 궁을 잡으면 승리한다. (게임이 끝난다)

### 장기판

- 10(가로) * 9(세로)

---

## 1.1 장기판 초기화

- [x] 고정 위치로 초기화(마 ↔ 상 위치 바꾸기 없음)

```

ㅁ 포 ㅁ ㅁ ㅁ ㅁ ㅁ 포 ㅁ
병 ㅁ 병 ㅁ 병 ㅁ 병 ㅁ 병
ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ
ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ ㅁ

ㅁ 포 ㅁ ㅁ ㅁ ㅁ ㅁ 포 ㅁ

차 상 마 사 ㅁ 사 상 마 차
```

## 1.2 기물 이동

- [x] 장기말은 색깔을 가지고 있다

#### 졸, 병

- [x] 앞이나 옆으로 한 칸 이동 가능
- [x] 후퇴 불가

#### 차

- [x] 가로, 세로 무한으로 이동 가능

#### 상

- [x] 선 한 칸 이동 + 대각성 두 칸 이동
- [x] 다른 기물이 이동 경로 사이에 있으면 이동 불가

#### 포

- [x] 다른 기물 하나를 넘어야만 이동 가능
- [x] 같은 포 끼리는 넘을 수 없음, 먹을 수 없음

#### 마

- [x] 선 한 칸 이동 + 대각선 한 칸 이동
- [x] 다른 기물이 이동 경로 사이에 있으면 이동 불가

### 이동 입력

- [x] (움직이기 전 위치) (기물이름) (움직일 위치)

ex) 03 마 84

## 2.1 기물 확장

### 사 & 궁

- [x] 선 한 칸 이동
- [x] 궁성 내부에서만 이동 가능 (안 -> 안)

### 궁성 대각 이동 가능 기물

- [x] 궁, 사, 차, 포 , 졸

### 장기의 점수 계산

- [x] 승패 출력
- [x] 남은 기물의 점수 합산하여 장기 점수 계산
    - 차: 13점
    - 포: 7점
    - 마: 5점
    - 상: 3점
    - 사: 3점
    - 졸: 2점
    - 합계: 72점 (한나라는 1.5점 추가)

## 2.2 DB 적용

- [x] 게임을 재시작하면 새로운 보드 또는 이전의 보드 불러옴
