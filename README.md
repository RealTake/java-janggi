# java-janggi

장기 미션 저장소

# 게임 규칙

- [x] 궁이 죽으면 남은 기물에 상관없이 패배한다.
- [x] 게임을 종료하게 된다면 기물들의 점수로 게임 결과를 판정한다. 점수가 많은 쪽이 승리한다.
- [x] 후수자인 한나라는 1.5점의 추가 점수가 부여된다.

## 장기판

- [x] 장기 기물들을 지정된 좌표에 배치한다.
- [x] 기물들은 장기판 밖으로 나갈 수 없다.
- [x] 장기판은 9 x 10으로 구성된다.
- [x] 가로열은 1번부터 9번까지 왼쪽에서 오른쪽으로 증가한다.
- [x] 세로열은 1번부터 10번까지 위에서 아래로 증가한다.
- [x] 움직일 수 없는 기물은 선택할 수 없다.
- [x] 기물은 지정된 좌표로만 움직일 수 있다.

## 장기 기물

- [x] 기물들은 각 움직일 수 있는 규칙이 있다.
- [x] 규칙에 의해 해당 좌표까지 움직일 수 있다.
- [x] 움직일 수 있는 좌표에 아군 기물이 있다면 해당 좌표로 갈 수 없다.
- [x] 각 나라별 차림을 설정할 수 있다.
- [x] 궁을 제외한 각 기물들은 점수가 존재한다.

### 병

- [x] 왼쪽, 오른쪽, 앞쪽으로 한 칸 움직일 수 있다.
- [x] 뒤(후퇴)로는 할 수 없다.
- [x] 끝에 도달하면 좌우로만 움직일 수 있다.
- [x] 궁성 내에서 적진에 방향으로 대각선 움직임이 허용된다.
- [x] 각 점수는 2점이다.

### 상

- [x] 앞으로 한 칸 이동 후 대각선 방향으로 두 칸 이동한다.
- [x] 이동하는 경로에 기물이 있다면 움직일 수 없다.
- [x] 각 점수는 3점이다.

### 마

- [x] 앞으로 한 칸 이동 후 대각선 방향 한 칸 이동
- [x] 이동 경로에 기물이 있으면 움직일 수 없다.
- [x] 각 점수는 5점이다.

### 포

- [x] 포는 포끼리 잡을 수 없다.
- [x] 중간에 포가 아닌 다른 기물이 있어야 그 방향으로 이동 가능하다.
- [x] 궁성 내에서만 대각선이 움직이는 게 가능하나 포가 아닌 다른 기물이 존재해야 한다.
- [x] 각 점수는 7점이다.

### 차

- [x] 가로 세로로 끝까지 갈 수 있다.
- [x] 이동 경로에 적군 기물이 있으면 움직일 수 없다.
- [x] 궁성 내에서만 대각선 움직임이 가능하다.
- [x] 각 점수는 13점이다.

### 사

- [x] 궁성 안에서만 움직일 수 있다.
- [x] 상하좌우, 대각선으로 한 칸 움직일 수 있다.
- [x] 각 점수는 3점이다.

### 궁

- [x] 궁성 안에서만 움직일 수 있다.
- [x] 상하좌우, 대각선으로 한 칸 움직일 수 있다.
- [x] 궁은 점수가 존재하지 않는다.

# 출력 사항

- [x] 각 나라에 맞추어서 색깔을 표현한다.
- [x] 빈 칸은 #으로 출력하도록 한다.
- [x] 각 기물에 맞는 한글로 표현한다.
- [x] 보드판에 좌표를 표시한다.

# 기능 사항

- [x] 방을 불러올지 새로할 지 선택한다.
    ```
    [안내] 방을 불러오시겠습니까?
    1. 방 불러오기
    2. 새로운 게임 시작하기
  ```
- [x] 방을 불러온다면 방 이름을 입력한다.
    ```
    [안내]방 이름을 입력해주세요
  ```
    - 방이 존재하지 않는다면 예외를 발생한다
        ```
        [ERROR] 해당 방이 존재하지 않습니다! 
      ```
    - 방 이름은 10자 이하로 입력해야 한다.
- [x] 새로운 게임을 시작한다면 방 이름을 설정한다.
    ```
  [안내]방 이름을 입력해주세요
  ```

- [x] 각 나라의 차림을 선택한다.
    ```
      [나라]의 차림을 숫자로 선택해주세요
      1. 상마상마
      2. 마상마상
      3. 마상상마
      4. 상마마상
  ```

- [x] 보드판을 출력한다.
    - [x] 먼저 해야할 나라를 출력한다.
        - [x] 이때 시작은 초나라부터 시작한다.
          ```
          [안내] 초나라의 차례입니다.
          ```

- [x] 각 차례가 할 옵션을 선택할 수 있다.
    - 기물 선택을 할 수 있을 때까지 옵션을 선택할 수 있다.
    ```
    [안내] 원하는 옵션을 선택해주세요!
    1. 기물 선택
    2. 점수 확인
    3. 저장
    4. 종료
  ```

- [x] 움직일 기물을 입력받는다.

    ```
    [안내] 기물의 좌표를 '세로,가로' 순으로 입력해주세요. (예: 3,5)
    ```

- [x] 다음과 같은 경우 예외를 발생하고 출력한다.
    - [x] 해당 기물이 존재하지 않음
      ```
      [ERROR] 해당 좌표에 기물이 존재하지 않습니다.
      ```

    - [x] 다른 나라의 기물
      ```
      [ERROR] 상대의 기물입니다.
      ```
    - [x] 움직이는 경우의 수가 존재하지 않는 기물
      ```
      [ERROR] 움직일 수 없는 기물입니다.
      ```

- [x] 기물을 입력받은 다음에는 움직일 좌표를 입력받는다.

    ```
    [안내] [해당 기물]이 움직일 좌표를 '세로,가로' 순으로 입력해주세요. (예: 3,5)
    ```

    - [x] 다음과 같은 경우에 예외가 발생된다.
        - [x] 움직일 수 없는 좌표
            ```
            [ERROR] 해당 좌표로 움직일 수 없습니다!
            ```

- [x] 움직인다면 데이터베이스에 저장된다

- [x] 기물이 움직였다면 다음 차례로 넘겨준다.
    ```
    [안내] 한나라의 차례입니다.
    ```

- [x] 현재 점수를 볼 수 있다.
    ```
    [안내] 현재 점수입니다. 
    한나라 : 10점
    초나라 : 20점
    ```

    - [x] 상대의 궁이 죽는다면 게임이 종료된다.

        ```
        [안내] 한나라가 이겼습니다!
        ```
- [x] 각 나라의 궁이 죽지 않고 종료된다면 점수를 비교하고 결과를 출력한다.
    ```
  [안내] 최종 점수입니다 
    한나라 : 73.5점
    초나라 : 72.0점
    [안내] 한나라가 이겼습니다!
    ```
