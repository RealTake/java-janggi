# java-janggi

장기 미션 저장소

### 실행 방법

프로젝트 내부 `docker-compose.yml` 파일이 존재하는 경로로 이동한 후, 다음 명령어를 실행해주세요.
 
<실행 시>
```
docker-compose -p janggi up -d
```

<종료 시>
```
docker-compose down
```
