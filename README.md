# java-janggi

> 장기의 보드판 그리고 각 기물과 움직임을 구현합니다.

## 구현할 기능 목록

- [x] 게임을 시작한다.
  - [x] 환영 메시지와 게임 규칙을 출력한다.
  - [x] `게임을 시작하시겠습니까? (y/n)` 문구를 출력한다.
    - n을 입력하는 경우 종료된다.
- [x] 보드판 초기화
- [x] 움직일 진영의 차례를 안내한다.
  - [x] `[{진영}의 차례입니다.]` 문구를 출력한다.
  - [x] `이동시킬 기물의 좌표와 도착 지점의 좌표를 입력해 주세요. 예) 03,13` 문구를 출력한다.
- [x] 게임을 진행한다.
  - [x] 각 차례마다 기물을 이동한다.
  - [x] 턴 진행 중 예외가 발생한 경우 재입력 받는다.
- [x] 게임을 종료한다.
  - [x] 장군이 쓰러질 경우 `{진영}의 장군이 쓰러졌습니다. {잔영}의 승리입니다.` 문구를 출력한다.
  - [x] 두 진영의 점수를 출력 후 프로그램을 종료한다.

## 게임 규칙

- 빨간색이 한, 초록색이 초
- 초부터 시작한다.

## 보드판

- [x] 9 X 10 사이즈이다.
- [x] 기물들의 좌표를 관리한다.
  - [x] 어떤 기물인지는 모른다.
  - [x] 기물은 영역 밖으로 넘어갈 수 없다.
  - [x] 기존 위치와 같은 위치로 기물을 이동할 경우 예외가 발생한다.

## 기물

- [x] 능력을 가진다.
- [x] 서로 협력한다.
- [x] 진영을 가진다.
  - 초, 한으로 구성된다.
- [x] 서로 잡을 수 있는 지 진영으로 비교한다.

### 마

- [x] 진영을 가진다.
- [x] 마는 직선으로 한 칸, 대각선으로 한 칸 움직일 수 있다.
  - [x] 아닐 경우 예외가 발생한다.
  - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.

### 상

- [x] 진영을 가진다.
- [x] 상은 직선으로 한 칸, 대각선으로 두 칸 움직일 수 있다.
  - [x] 아닐 경우 예외가 발생한다.
  - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.

### 차

- [x] 진영을 가진다.
- [x] 차는 상하좌우 무제한으로 움직일 수 있다.
  - [x] 상화좌우 방향이 아닌 경우 예외가 발생한다.
  - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.
- [x] 차는 궁성 범위 안에서 대각선으로 움직일 수 있다.

### 포

- [x] 진영을 가진다.
- [x] 포는 상하좌우로 움직일 수 있다.
  - [x] 포와 잡을 기물 사이에 단 하나의 기물이 있어야 한다.
    - [x] 그 기물은 포일 수 없다.
  - [x] 아닐 경우 예외가 발생한다.
- [x] 포는 궁성 범위 안에서 대각선으로 움직일 수 있다.

### 군인

- [x] 진영을 가진다.
  - 초 진영은 졸로 표현한다.
  - 한 진영은 병으로 표현한다.
- [x] 병은 앞 또는 양 옆으로 한 칸만 움직일 수 있다.
  - [x] 뒤로 움직일 경우 예외가 발생한다.
- [x] 군인은 궁성 범위 안에서 대각선으로 움직일 수 있다.

### 장군

- [x] 진영을 가진다.
- [x] 장군은 궁성에서 벗어날 수 없다.
  - [x] 궁성 밖으로 벗어나는 경우 예외가 발생한다.
- [x] 장군은 상하좌우, 대각선으로 한 칸 움직일 수 있다.
  - [x] 한 칸 이상 움직이는 경우 예외가 발생한다.

### 사

- [x] 진영을 가진다.
- [x] 사는 궁성에서 벗어날 수 없다.
  - [x] 궁성 밖으로 벗어나는 경우 예외가 발생한다.
- [x] 사는 상하좌우, 대각선으로 한 칸 움직일 수 있다.
  - [x] 한 칸 이상 움직이는 경우 예외가 발생한다.

## 보드판 초기화 객체

- [x] 특정 좌표에 특정 기물을 올려둔다.

## 좌표

- [x] x와 y를 가진다.
- [x] 비교할 수 있다.
  - [x] 같은 수평선상에 있는지 확인한다.
  - [x] 같은 수직선상에 있는지 확인한다.
  - [x] 좌표의 차를 계산한다.
    - x, y 모두 가능하다.
  - [x] 같은 좌표에 있는지 확인한다.

## 명령어

- [x] `xy,x'y'` 형식으로 움질일 위치를 입력 받는다.

## 점수

- [x] 기물의 점수를 계산하여 승패와 관계 없이 모두 노출한다.
  - 기물의 점수는 아래와 같다.
    ````
    차: 13
    포: 7
    마: 5
    상: 3
    사: 3
    졸·병: 2
    ````

### 데이터베이스 연결

- [x] 애플리케이션을 재시작하더라도 이전에 하던 장기 게임을 다시 시작할 수 있어야 한다.
