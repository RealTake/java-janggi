# java-janggi

> 장기의 보드판 그리고 각 기물과 움직임, 점수 계산을 구현합니다.

## 구현할 기능 목록

- [x] 게임을 시작한다.
    - [x] 환영 메시지와 게임 규칙을 출력한다.
    - [x] `게임을 시작하시겠습니까? (y/n)` 문구를 출력한다.
        - n을 입력하는 경우 종료된다.
- [x] 보드판 초기화
- [x] 움직일 진영의 차례를 안내한다.
    - [x] `[{진영}의 차례입니다.]` 문구를 출력한다.
    - [x] `이동시킬 기물의 좌표를 입력해 주세요. 예) 03` 문구를 출력한다.
    - [x] `기물을 이동할 위치의 좌표를 입력해 주세요. 예) 13` 문구를 출력한다.
- [x] 게임을 진행한다.
    - [x] 각 차례마다 기물을 이동한다.
    - [x] 턴 진행 중 예외가 발생한 경우 재입력 받는다.
    - [x] 현재 점수를 출력한다.
    - [x] 왕이 잡히면 게임을 종료한다.

## DB 구성 방법

1. [Docker Download](https://docs.docker.com/get-started/get-docker/)를 통해 각 OS에 맞게 Docker를 설치합니다.
2. 프로젝트 루트 디렉토리에서 `docker-compose up -d` 명령어를 실행합니다.
3. `docker-compose ps` 명령어를 통해 컨테이너가 정상적으로 실행되었는지 확인합니다.
4. 종료할 때는 `docker-compose down` 명령어를 실행합니다.

## 게임 모드

- [x] 오프라인 모드
    - [x] 매번 새로운 게임을 시작한다.
- [x] 온라인 모드 (DB 연결 필요)
    - [x] 게임 종료 후 이전 게임을 불러온다.

## 게임 규칙

- 빨간색이 한나라, 초록색이 초나라이다.
- 좌표계는 고정이다.
    - 각 축의 번호는 0부터 시작한다.
    - 시작 지점은 왼쪽 하단(0, 0)이다.
- [x] 게임은 초나라부터 시작된다.
    - [x] 후공 한나라는 전체 점수에서 1.5점을 더한다. (후공 배려)
- [x] 장군이 잡히면 게임이 종료된다.

## 보드판

- [x] 9 X 10 사이즈이다.
- [x] 기물들의 좌표를 관리한다.
    - [x] 어떤 기물인지는 모른다.
    - [x] 기물은 영역 밖으로 넘어갈 수 없다.
    - [x] 기존 위치와 같은 위치로 기물을 이동할 경우 예외가 발생한다.

### 궁

- [x] 궁은 3 X 3 사이즈이다.
    - [x] 궁에는 각 모서리와 중간을 잇는 대각선이 있다.
- [x] 왕과 사는 궁 안에서만 움직일 수 있다.

## 기물

- [x] 능력을 가진다.
- [x] 서로 협력한다.
- [x] 진영을 가진다.
    - 초나라와 한나라로 나뉜다.
- [x] 서로 잡을 수 있는 지 진영으로 판단한다.
- [x] 각 기물은 점수를 가진다.
    - [x] 포: 7점
    - [x] 차: 13점
    - [x] 상: 3점
    - [x] 마: 5점
    - [x] 왕: 0점
    - [x] 사: 3점
    - [x] 졸, 병: 2점

### 마

- [x] 진영을 가진다.
- [x] 마는 직선으로 한 칸, 대각선으로 한 칸 움직일 수 있다.
    - [x] 아닐 경우 예외가 발생한다.
    - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.

### 상

- [x] 진영을 가진다.
- [x] 상은 직선으로 한 칸, 대각선으로 두 칸 움직일 수 있다.
    - [x] 아닐 경우 예외가 발생한다.
    - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.

### 차

- [x] 진영을 가진다.
- [x] 차는 상하좌우 무제한으로 움직일 수 있다.
    - [x] 상화좌우 방향이 아닌 경우 예외가 발생한다.
    - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.

### 포

- [x] 진영을 가진다.
- [x] 포는 상하좌우로 움직일 수 있다.
    - [x] 포와 잡을 기물 사이에 단 하나의 기물이 있어야 한다.
        - [x] 그 기물은 포일 수 없다.
    - [x] 아닐 경우 예외가 발생한다.

### 졸

- [x] 진영을 가진다.
    - 초나라이다.
- 뒤로는 움직일 수 없으며, 앞 또는 양 옆으로 1칸만 움직일 수 있다.
    - 좌표계는 고정이므로, 졸에게 뒤는 y가 감소하는 방향이다.

### 병

- [x] 진영을 가진다.
    - 한나라이다.
- 뒤로는 움직일 수 없으며, 앞 또는 양 옆으로 1칸만 움직일 수 있다.
    - 좌표계는 고정이므로, 병에게 뒤는 y가 증가하는 방향이다.

### 왕

- [x] 진영을 가진다.
- [x] 왕은 직선으로 한 칸 움직일 수 있다.
    - [x] 아닐 경우 예외가 발생한다.
    - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.
- [x] 왕은 궁 안에서만 움직일 수 있다.

### 사

- [x] 진영을 가진다.
- [x] 사는 직선으로 한 칸 움직일 수 있다.
    - [x] 아닐 경우 예외가 발생한다.
    - [x] 다른 기물을 넘어 이동할 경우 예외가 발생한다.
- [x] 사는 궁 안에서만 움직일 수 있다.

## 보드판 초기화 객체

- [x] 특정 좌표에 특정 기물을 올려둔다.

## 좌표

- [x] x와 y를 가진다.
- [x] 비교할 수 있다.
    - [x] 같은 수평선상에 있는지 확인한다.
    - [x] 같은 수직선상에 있는지 확인한다.
    - [x] 좌표의 차를 계산한다.
        - x, y 모두 가능하다.
    - [x] 같은 좌표에 있는지 확인한다.
    - [x] 좌표계를 기준으로 뒤에 있는지 확인한다.
    - [x] 한 칸만 움직일 수 있는지 확인한다.

## 좌표 입력

- [x] `xy'` 형식으로 좌표를 입력 받는다.
