# java-janggi

장기 미션 저장소

## 기능 요구사항

### 보드판

- 크기: 9 * 10
- 가로선은 위에서부터 아래로 1부터 0까지(1,2,3..,9,0) 표기
- 세로줄은 각각 왼쪽에서 오른쪽으로 1부터 9까지(1,2,3..,9) 표기

### 기물

- 마
    - 한 칸 전진 후 대각선으로 움직인다.
    - 진행 경로에 기물이 있으면 움직이지 못한다.
    - 마를 잡으면 5점을 획득한다.
- 상
    - 한 칸 전진 후 두 칸 대각선으로 움직인다.
    - 진행 경로에 기물이 있으면 움직이지 못한다.
    - 상을 잡으면 3점을 획득한다.
- 차
    - 수직, 수평으로 움직인다.
    - 기물을 뛰어넘을 수 없다.
    - 차를 잡으면 13점을 획득한다.
- 포
    - 기물 하나만 뛰어넘을 수 있다.
    - 포는 뛰어넘을 수 없고, 포를 잡을 수 없다.
    - 포를 잡으면 7점을 획득한다.
- 병
    - 뒤를 제외하고 한 칸 움직인다.
    - 병을 잡으면 2점을 획득한다.
- 장군
    - 궁 내에서 선을 따라 한 칸 움직인다.
    - 장군을 잡으면 게임이 종료된다.
- 사
    - 궁 내에서 선을 따라 한 칸 움직인다.
    - 사를 잡으면 3점을 획득한다.

### 궁성

- 한나라의 궁성 영역 좌표
    - (1,4), (1,5), (1,6), (2,4), (2,5), (2,6), (3,4), (3,5), (3,6)
- 초나라의 궁성 영역 좌표
    - (7,4), (7,5), (7,6), (8,4), (8,5), (8,6), (9,4), (9,5), (9,6)
- 궁성 내 기물의 움직임
    - 차, 포, 궁, 사, 병은 궁성의 대각선 경로를 이동할 수 있다.
    - 궁, 사는 궁성 영역 밖으로 벗어날 수 없다.

## 기능 구현

### 장기 게임

- [x] 플레이어 턴을 관리한다.
- [x] 기물을 움직인다.
- [x] 승패를 결정한다.
- [x] 이전 게임을 불러온다.
    - 기물 배치
        - [x] 장기판 위 기물 배치 정보를 DB에 저장한다.
        - [x] 장기판 위 기물 배치 정보를 DB에서 불러온다.
        - [x] 장기판 위 기물 배치 정보를 DB에서 업데이트한다.
    - 턴
        - [x] 현재 턴 정보를 DB에 저장한다.
        - [x] 현재 턴 정보를 DB에서 불러온다.
        - [x] 현재 턴 정보를 DB에서 업데이트한다.
- [x] 게임방에 입장할 수 있다.
    - [x] 게임방을 생성한다.
    - [x] 게임방을 조회한다.
    - [x] 게임이 종료되면 게임방을 삭제한다.

### DB

- 저장이 필요한 값: 기물의 좌표, 어떤 기물인지, 어떤 나라인지, 현재 턴
    - DB: pieces 테이블
    - `id`, `piece_name`, `x`, `y`, `country`, `game_id`
    - DB: games 테이블
    - `id`, `name`, `current_turn`

### 보드판

- [x] 좌표와 해당 좌표에 있는 기물을 가진다.
- [x] 기물의 목적지가 유효한 좌표인지 검사한다.
- [x] 궁성의 영역 규칙을 적용한다.
- [x] 각 나라의 점수를 계산한다.

### 기물

- [x] 규칙에 따라 움직인다.
- [x] 기물마다 점수를 가진다.

## 실행 가이드

### db 환경 설정

1. 터미널에서 java-janggi/docker 디렉토리로 이동합니다.
2. 다음 명령어를 통해 MySQL 컨테이너를 실행합니다.
   ```bash
   docker compose -p janggi up -d
    ```

- 주의 사항
    - schema.sql 파일은 컨테이너가 처음 생성될 때만 자동 실행됩니다.
    - DDL을 다시 실행하고 싶다면, 볼륨까지 삭제하고 컨테이너를 재생성해야 합니다
      ```bash
      `docker compose -p janggi down -v`
      `docker compose -p janggi up -d`
      ```
