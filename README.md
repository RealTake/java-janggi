# java-janggi

장기 미션 저장소

---

## 실행 방법

콘솔에서 아래 명령어 실행 시 docker-compose.yml 파일에 따라 DB 환경 설정이 진행됩니다!
```
docker-compose -p janggi up -d
```

---

## 기능 구현 목록

### 초기화
- [x] 사용자로부터 상차림을 입력받습니다.
- [x] 상차림 정보를 기준으로 장기판을 생성합니다.

### 이동

- [x] 초 진영의 움직일 기물 종류, 출발 좌표, 도착 좌표를 입력받습니다.
  - [x] 검증 : 입력받은 출발 좌표와 도착 좌표가 장기판 범위를 벗어나는 경우 예외를 발생시킵니다.
  - [x] 검증: 선택한 기물이 출발 좌표에 존재하지 않으면 예외를 발생시킵니다.
  - [x] 검증 : 도착 좌표에 아군 말이 존재하는 경우 예외를 발생시킵니다. or 이동 시키지 않습니다.
  
- [x] 선택한 기물이 좌표에 존재하면 출발 좌표에서 도착 좌표 사이의 모든 경로 좌표를 계산합니다.
  - [x] 검증 : 선택한 기물의 시작 좌표와 도착 좌표 사이에 장애물이 존재하는지 확인합니다.
    - [x] 일반 말 : 장애물이 존재하면 예외를 발생시킵니다.
    - [x] 포 : 2개 이상의 장애물이 존재하거나, 장애물이 1개만 존재하는 경우 예외를 발생시킵니다.
    
- [x] 도착 좌표로 기물을 이동 시킵니다.
  - [x] 도착 좌표에 아무 기물도 없는 경우 기물을 이동시킵니다.
  - [x] 도착 좌표에 적팀 기물이 존재하는 경우 해당 기물은 죽습니다.

- 기물 이동
  - [x] 기물 이동 시, 이동 경로에 장애물이 존재하는 경우 이동할 수 없습니다. 
- 왕
  - [x] 상 하 좌 우 대각선 모두 한칸 씩 이동합니다.
- 사
  - [x] 상 하 좌 우 대각선 모두 한칸 씩 이동합니다.
- 졸/병
  - [x] 바라보는 방향으로 전진 및 좌 우 이동합니다.
- 마
  - [x] 상 하 좌 우 한칸 이동 후 대각선으로 한 번 이동합니다.
- 상
  - [x] 상 하 좌 우 한칸 이동 후 대각선으로 두 번 이동합니다.
- 차
  - [x] 상 하 좌 우 칸수 제한 없이 이동합니다.
- 포
  - [x] 상 하 좌 우 칸수 제한 없이 이동합니다.
  - [x] 포는 포를 넘을 수 없습니다.
  - [x] 포는 포를 잡을 수 없습니다.

### 게임 종료 조건 확인
- [x] 보드에 살아있는 왕이 2개 미만으로 존재하는 경우 게임이 종료됩니다.
- [x] 게임 종료 시 살아있는 왕이 존재하는 진영이 승리하며 이를 보여줍니다.

### 게임 점수 계산
- [x] 매 턴마다 각 진영의 살아있는 기물을 기준으로 총점을 계산합니다.
- [x] 매 턴, 각 진영 점수를 보여줍니다.

### 궁성 기능 구현 

- [x] 각 진영 별 궁과 궁성 영역을 구분합니다.
- [x] 현재 기물 위치가 궁(palace) 또는 궁성(fortress) 영역 내에 위치하는지 확인합니다.
- [x] 궁 또는 궁성에 위치하는 경우 기물에 이동 가능한 움직임들을 추가합니다.
- [x] 궁 또는 궁성에 위치하는 경우 이동 가능한 움직임을 골라냅니다.
- [x] 현재 위치가 궁 또는 궁성 내에 위치하는 경우 궁성에 맞는 움직임이 추가됩니다.
- [x] 왕, 사의 경우 궁성 밖으로 이동할 수 없습니다.
