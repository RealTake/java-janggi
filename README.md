# java-janggi

## 기물

- [x] 기물은 위치를 갖는다.
    - [x] X축 (0-8), Y축(0-9)로 구현한다.
- [x] 기물은 팀을 갖는다.
    - [x] 팀은 한나라, 초나라로 나뉘어 진다.
- [x] 기물은 움직인다.
    - [x] 다른 기물과 상관없이 갈 수 있는 경로를 얻는다.

## 위치

- [x] 위치는 x축 기준 0-8, y축 기준 0-9만 가능하다.

## 기물 모음

- [x] 다른 기물들의 위치를 이용해 각 특징에 따라 최종적으로 갈 수 있는 경로를 선별한다.
- [x] 기물은 다른 기물을 잡을 수 있다.
    - [x] 기물은 같은 팀을 잡지 못하고, 오직 다른 팀만 잡을 수 있다.

## 경로

- [x] 위치를 받아, 경로에 포함되는지 확인한다.
- [x] 위치를 받아, 경로의 최종 도착지인지 확인한다.

## 보드

- [x] 보드를 초기화 한다.
    - [x] 기물을 초기 위치에 둔다.
        - [x] 마상마상, 마상상마, 상마상마, 상마마상을 고른다. 해당 입력에 따라 장기판을 구성한다.
        - [ ] 각 팀마다 다른 상차림을 선택할 수 있게 한다.
- [x] 게임을 진행한다.

## 턴

- [x] 한나라, 초나라 순으로 게임을 진행한다.
- [x] 턴을 바꿀 수 있다.

## 예외사항

- [x] 예외 발생시, 재입력받는다.

- [x] 사용자가 게임 진행여부 입력시, Y, N 이외의 입력을 한 경우
- [x] 기물이 없는 위치를 선택할 경우
- [x] 자신의 기물이 아닌 위치를 선택했을 경우
- [x] 이동할 수 없는 위치를 고른 경우

## 궁성

기물이 궁성 영역에 들어왔을 때, 기물에 따라 다른 움직임을 한다.

- [x] 왕, 사 는 궁성 외부로 나갈 수 없다.
- [x] 차는 대각선으로 직진할 수 있다.
    - [x] 차는 (3,0) <-> (5,2), (5,0) <-> (3,2) 로 이동 가능하다
- [x] 포는 대각선 방향으로 기존 이동 규칙을 유지할 수 있다.
    - [x] 포는 뛰어넘을 기물이 있다면, (3,0) <-> (5,2), (5,0) <-> (3,2) 로 이동 가능하다
- [x] 마, 상 은 대각선 이동을 할 수 없다.

## 게임 승패 결정 조건

1. 상대방 왕이 죽은 경우
2. 30턴이 지난 후, 점수를 계산해 승리를 계산한다.
3. 종료를 누르면, 해당 턴의 사람이 패배한다.
