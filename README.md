# 자바로 구현한 코리안 체스 "장기" 시뮬레이터

### 소개
안녕하세요. 장기 기물들을 장기 규칙에 맞게 이동시킬 수 있는 시뮬레이터 애플리케이션입니다.
- 설계와 관련된 사항은 `DESCRIPTION.md`를 읽어주세요!

### 필독!
데이터베이스 연결을 위해 아래의 사항을 수행해주셔야 합니다!

```shell
# project directory에서, 수행
cd docker

# /docker 위치에서 수행
docker-compose -p chess up -d
```

### 기능 1. 기물의 이동
각 기물들은 장기 규칙에 따라서 이동시킬 수 있습니다.  
장기 기물 규칙은 다음을 참고해주세요 !  
- [기물의 이동 규칙](https://namu.wiki/w/%EC%9E%A5%EA%B8%B0/%EA%B8%B0%EB%AC%BC%20%EB%B0%8F%20%ED%96%89%EB%A7%88%EB%B2%95)

### 기능 2. 시뮬레이션의 종료
장군이 모두 살아있지 않다면 게임이 종료된다.
게임이 종료된 후 우승 국가를 출력해준다.


### 기능 3. 점수 계산
각 플레이어들은 기물을 잡을 때 점수가 추가된다.
게임이 종료된 후 우승자의 점수를 출력해준다.
