# java-janggi

장기 미션 저장소

## 입력

- [x] 이동시킬 장기의 위치를 좌표로 입력한다.
- [x] 이동할 위치를 좌표로 입력한다.

## 도메인

### 장기 게임

- [x] 장기 게임을 실행한다.
- [x] 한 차례식 턴을 바꾸며 게임을 진행한다.
- [x] 왕이 죽으면 게임이 종료된다.

### 위치

- [x] 행과 열의 위치 정보를 가진다.
- [x] 장기판의 범위를 초과하면 예외를 발생한다.
- [x] 대각선 한칸을 판단한다.
- [x] 한칸 (앞,옆,뒤)을 판단한다.

### 장기판

- [x] 장기판은 기물과 위치를 가진다.
- [x] 궁성의 영역을 가진다.
    - [x] 궁성의 영역은 초,한 두개다
- [x] 목적지 까지의 경로를 검사한다.
    - [x] 경로에 장애물이 있다면 예외를 던진다.
- [x] 기물들의 위치를 업데이트 한다.
- [x] 선택한 위치에 기물이 존재하지 않으면 예외가 발생한다.
- [x] 기물을 잡을 수 있다.
- [x] 각 팀의 총 기물의 점수를 계산한다.

## 궁성

- [x] 궁성 영영을 가진다.
- [x] 현재 위치가 궁성에 포함되는지 판별한다.

### 기물

- [x] 기물은 타입, 팀을 가진다.
- [x] 기물은 본인의 움직임을 계산할 수 있다.
- [x] 기물이 이동할 수 없는 위치라면 예외를 던진다.
- [x] 자신이 가야하는 경로의 위치를 계산한다.
- [x] 자신이 가야하는 경로의 위치에 장애물이 있는지 검사한다.
- [x] 같은 팀의 기물을 잡을 수 없다.
- [x] 상대팀과 자신의 팀을 구별한다.

#### 왕(king)

- [x] 궁성 안에서만 움직일 수 있다.
    - [x] 궁성을 이탈하면 예외가 발생한다.
- [x] 상하좌우, 간선(대각선)이동이 가능하다.
- [x] 왕이 다른 기물에게 잡히면 게임이 종료된다.

#### 사(Guard)

- [x] 궁성 안에서만 움직일 수 있다.
    - [x] 궁성을 이탈하면 예외가 발생한다.
- [x] 상하좌우, 간선(대각선)이동이 가능하다.

#### 상(Elephant)

- [x] 궁성 안에서 움직임의 변화가 없다.
- [x] 한칸 전진 후, 대각선으로 두 칸 이동이 가능하다.
    - [x] 한칸 전진 시, 궁성의 대각선은 제외한다.
- [x] 이동 경로에 멱(장애물)이 있다면 이동할 수 없다.

#### 마(Horse)

- [x] 궁성 안에서 움직임의 변화가 없다.
- [x] 한칸 전진 후, 대각선으로 한 칸 이동이 가능하다.
    - [x] 한칸 전진 시, 궁성의 대각선은 제외한다.
- [x] 이동 경로에 멱(장애물)이 있다면 이동할 수 없다.

#### 차(Chariot)

- [x] 가로막는 다른 기물만 없다면 상하좌우로 가고 싶은 곳으로 이동할 수 있다.(장기판의 범위 내에서 한방향으로 쭉 이동이 가능하다)
- [x] 궁성의 영역에서는 대각선을 타고 몇칸이든 가고 싶은 곳으로 질주할 수 있다.

#### 포(Cannon)

- [x] 차와 같이 상하좌우 이동이 가능하다.
    - [x] 궁성의 대각선 이동이 가능하다.
    - [x] 궁성 안에서도 반드시 하나의 기물을 뛰어 넘어야 한다.
- [x] 반드시 하나의 기물을 뛰어 넘어야 한다.
    - [x] 하나의 기물만을 넘어야 한다.(2개 이상x)
    - [x] 자기팀과 상대팀을 가리지 않는다.
- [x] 같은 포끼리는 넘을 수 없다.

#### 졸/병(Pawn/Soldier)

- [x] 졸(초나라)
- [x] 병(한나라)
- [x] 한칸 전진과 좌우 이동이 가능하다.
    - [x] 자신의 진영을 기준으로 후퇴를 한다면 예외가 발생한다.
- [x] 상대편의 궁성에 진입하면 전진, 좌우, 대각선 이동이 가능하다.
    - [x] 대각선 이동시에도 전진(10시, 1시)방향만 가능하다.

### 기물 타입

- [x] 기물 타입은 기물의 타입을 가진다.
- [x] 각각의 기물에 대한 점수를 가진다.

### 점수

- [x] 점수를 가진다.

### 게임 상태

- [x] 진행, 종료 상태를 가진다.

### 팀

- [x] 팀은 초나라 한나라중 하나의 상태를 가진다.
- [x] 턴을 할 수 있다.

### 초기 기물 위치 생성기

- [x] 한나라, 초나라 기물을 생성한다.

## 출력

- [x] 불러온 게임, 새로운 게임 정보 출력
- [x] 장기판 세팅 출력
- [x] 장기 이동 결과 출력
- [x] 점수 현황 출력

## 데이터베이스

- [x] 게임의 진행상태를 db에 저장하여 프로그램을 재시작해도 이전에 하던 장기 게임을 다시 시작할 수 있다.
- [ ] 게임이 종료되면 db에 저장되어 있던 테이블 정보를 삭제한다.

### 게임 테이블

- [x] 게임을 생성하고 진행상태를 저장할 수 있다.

### 피스 테이블

- [x] 피스의 타입, 팀, 위치 값, 게임_id 를 가지고 피스의 상태를 저장할 수 있다.

## 실행 흐름

1. 이전에 진행하던 게임이 db에 있다면 불러오고, 아니라면 새로운 게임을 생성한다.
2. 장기판을 세팅하고 출력한다.
3. 이동을 원하는 기물과 위치를 입력한다.
    - 없는 기물을 입력한 경우 예외처리
    - 기물의 위치를 잘못 입력한 경우 예외처리
4. 입력받은 위치까지의 경로를 확인한다.
    - 잘 못된 입력을 할 경우 예외처리 후 재입력 한다.
5. 입력받은 위치에 기물이 있는 경우 해당 기물을 탈락 처리한다.
6. 탈락된 기물의 팀의 점수 총합계에서 탈락된 기물 만큼의 점수를 제외하고 출력한다.
7. 왕이 잡히면 게임이 종료된다.
8. 게임 종료 안내 메세지와 점수를 출력한다.
