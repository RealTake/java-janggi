# java-janggi

장기 미션 저장소

## 기능 요구 사항
- 게임 시작 시 보드 초기화를 진행한다.
  - 초나라(플레이어1) 플레이어는 번호로 제시된 마와 상의 4가지 유형(ex, 원앙상, 귀마, 양귀마, 양귀상)의 선택지를 출력한다.
  - 초나라 사용자로부터 제시된 선택지 번호를 입력 받는다.
  - 한나라(플레이어2) 플레이어는 번호로 제시된 마와 상의 4가지 유형(ex, 원앙상, 귀마, 양귀마, 양귀상)의 선택지를 출력한다.
  - 한나라 사용자로부터 제시된 선택지 번호를 입력 받는다.
- 게임이 시작되면, 플레이어는 턴을 번갈아 진행한다.(턴의 내용은 아래와 같다)
  - 매 턴 시작시 보드 상태를 갱신하여 출력한다.
  - 플레이어로부터 이동할 말의 이름을 입력받는다.
    - 입력받은 말의 이름이 유효한지 검증한다.
    - 입력받은 말이 살아있는지 검증한다.
  - 플레이어로부터 이동할 위치의 좌표를 입력받는다.
    - 입력받은 좌표가 장기 판의 크기를 벗어나지 않는지 검증한다.
    - 입력받은 좌표가 이동하려는 말의 이동 규칙에 맞는지 검증한다.
  - 입력 받은 위치로 기물을 이동 시킨다.
    - 이동한 위치에 상대방의 기물이 있다면, 상대방의 기물을 잡는다.
- 궁이 잡히면 게임이 종료된다.
  - 궁을 잡은 플레이어의 승리 메시지를 출력한다.(ex,"초나라 승리!", "한나라 승리!")

### 기물 이동 규칙 요구사항
**졸(Soldier)의 이동 방법**
1. 이동 방향 및 거리
   - [x] 전진 또는 좌/우 가로 방향으로 1칸만 이동 가능하다.
   - [x] 후진은 불가능하다.
2. 진영 별 전진 방향
   - [x] 초나라는 y의 좌표가 감소하는 방향이 전진이다.
   - [x] 한나라는 y의 좌표가 증가하는 방향이 전진이다.
3. 이동 제한
   - [ ] 목적지에 아군 기물이 있는 경우 이동할 수 없다.

**궁과 사(King)의 이동 방법**

1. 이동 방향 및 거리
   - [x] 가로, 세로, 대각선 방향으로 1칸만 이동 가능하다.
2. 이동 제한
   - [x] 목적지에 아군 기물이 있는 경우 이동할 수 없다.
   - [ ] 궁성 밖으로 나갈 수 없으며, 궁성 내부에서도 대각선 방향으로 이동할 때는 궁성 중심을 지나는 대각선만 가능하다.

**마(Horse)의 이동 방법**

1. 이동 방향 및 거리
   - [x] 1단계 : 상,하,좌,우 1칸 이동한다.
   - [x] 2단계 : 1단계 이동 방향을 기준으로 대각선 앞 방향으로 1칸 이동한다.
2. 이동 제한
   - [x] 1단계에서 기물이 있는 경우 이동할 수 없다.
   - [x] 2단계에서 아군 기물이 있는 경우 이동할 수 없다.
   - [x] 2단계에서 상대 기물이 있는 경우 잡으면서 이동한다.


**차(Chariot)의 이동 방법**

1. 이동 방향 및 거리 
   - [x] 상,하,좌,우 이동이 가능하다.
2. 이동 제한
   - [x] 이동하려는 경로에 기물이 없어야 한다.

### 고급 기능 요구 사항(추후 개발 예정)
- [ ] 플레이어가 이동 가능한 말의 이동 경로를 출력한다.
