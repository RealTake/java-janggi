# :sparkling_heart: java-janggi 프로젝트 소개

웹 백엔드 7기 레벨1 장기 미션을 구현한 프로젝트입니다.

# :dart: 프로그램 실행 방법

- docker를 활용해 프로덕션 환경의 DB 실행

    ```
    cd docker/production
    docker-compose up
    ```

- 프로그램 실행 및 활용

- 프로그램 종료 후에는 생성했던 DB 정리
    ```
    docker-compose down
    ```

# :dart: 테스트 실행 방법

- docker를 활용해 테스트 환경의 DB 실행

    ```
    cd docker/test
    docker-compose up
    ```

- 테스트 수행

- 테스트 종료 후에는 생성했던 DB 정리
    ```
    docker-compose down
    ```

# :dart: 구현 기능 목록

## :rocket: 게임 메뉴

- [x] 게임메뉴 선택
    ```
    재밌는 장기를 해봐요!
    1. 새로 하기
    2. 이어서 하기
    1
    ```
- [x] 새로 하기 메뉴를 선택할 경우 새 게임을 시작할 수 있다.
- [x] 이어하기 메뉴를 선택할 경우 기존의 게임목록을 출력해준다.
    ```
    이어서 할 게임을 선택해주세요.
    0. game1
    1. game2
    2. game2
    3. 뒤로가기
    ...
    1
    ```
- [x] 이어서 할 게임을 선택한 경우 해당 게임을 이어서 할 수 있다.

## :rocket: 게임 시작

- [x] 생성할 게임 이름 입력
    ```
    생성할 게임의 이름을 입력해주세요.(50자 이내)
    5월31일 두번째 게임
    ```

- [x] 게임시작 문구 출력

    ```
    장기 게임을 시작하겠습니다!
    ```

## :rocket: 턴 진행

- [x] 이번 턴에 수행할 행동 선택
    ```
    초의 턴입니다. 이번턴에 진행할 행동을 정해주세요!
    1. 기물 움직이기
    2. 한수 쉬어주기
    3. 나중에 이어서 하기
    4. 승패를 가르고 종료하기
    ```
- [x] 초을 선수로 한과 번갈아가며 턴을 반복할 수 있다.

## :rocket: 기물 이동

- [x] 초을 선수로 한과 번갈아가며 턴을 반복할 수 있다.

    ```
    이동할 장기말의 좌표 입력해주세요.
    0,1
    장기말의 좌표를 입력해주세요
    0,2
    ```

    - [x] 예외처리 - 좌표가 장기판의 범위를 벗어난 경우

    ```
    [ERROR] x좌표는 0~8, y좌표는 0~9 사이로 입력해주세요.
    ```

    - [x] 예외처리 - 이동할 장기말이 입력된 좌표에 없는 경우

    ```
    [ERROR] 해당 위치에 이동할 말이 존재하지 않습니다.
    ```
    - [x] 예외처리 - 장기말의 이동 규칙에 어긋난 경우

    ```
    [ERROR] 장기말의 이동 규칙을 확인해주세요.
    ```

    - [x] 예외처리 - 장애물로 인해 이동이 불가능한 위치인 경우

    ```
    [ERROR] 장애물로 인해 이동이 불가능합니다.
    ```

- [x] 각 장기말 별로 이동을 구현할 수 있다.
    - [x] 궁
        - [x] 8방향으로 1칸씩 이동
        - [x] 궁안에서만 움직일 수 있음
    - [x] 사
        - [x] 8방향으로 1칸씩 이동
        - [x] 궁안에서만 움직일 수 있음
    - [x] 차
        - [x] 동서남북 방향으로 끝까지 이동 가능
        - [x] 장애물을 넘지 못함
        - [x] 궁안에서 대각선을 타고 이동 가능
    - [x] 포
        - [x] 동서남북 방향으로 끝까지 이동 가능
        - [x] 혼자서는 움직일 수 없음
        - [x] 장애물이 있어야지 넘어서 이동 가능
        - [x] 포는 포를 잡을 수 없음
        - [x] 포는 서로의 장애물이 될 수 없음
        - [x] 궁안에서 대각선을 타고 이동 가능
    - [x] 마
        - [x] 동사남북 방향으로 1칸이동 후 대각선을 1칸이동
        - [x] 장애물은 넘지 못한다.
        - [x] 궁안에서는 첫번째 움짐임에서는 대각선 이동 불가능
    - [x] 상
        - [x] 동사남북 방향으로 1칸 이동 후 같은 방향의 대각선으로 2칸 이동
        - [x] 궁안에서는 첫번째 움짐임에서는 대각선 이동 불가능
    - [x] 졸
        - [x] 앞과 양옆으로 1칸씩 이동 가능
        - [x] 뒤로는 이동 불가능
        - [x] 궁안에서 대각선 이동 가능
    - [x] 병
        - [x] 앞과 양옆으로 1칸씩 이동 가능
        - [x] 뒤로는 이동 불가능
        - [x] 궁안에서 대각선 이동 가능

- [x] 목적지에 상대 장기말이 있을 경우 잡을 수 있다.
- [x] 각 진영의 점수를 계산할 수 있다.

- [x] 기물 이동 후의 게임상태 출력
    - [x] 각 진영의 점수 출력
      ```
      초의 점수 : 20점
      한의 점수 : 20.5점
      ```

## :rocket: 한수 쉬기

## :rocket: 나중에 이어서 하기

- [x] 게임저장 문구를 출력할 수 있다
    ```
    "열실히 게임해요" 게임을 저장하고 종료합니다. 나중에 이어서 하실 수 있습니다.
    ```
- [x] 현재 게임을 저장하고 나중에 이어서 할 수 있다.

## :rocket: 승패를 가르고 종료하기

- [x] 게임 종료 조건
    - [x] 상대방의 궁을 잡으면 자동으로 종료
    - [x] 턴 진행중에 게임 종료선언을 할 경우

- [x] 승패 결정
    - [x] 상대방의 궁을 잡으면 승리
    - [x] 궁을 잡히면 패배
    - [x] 게임을 중간에 그만둘 경우 점수로 승패 결정

- [x] 게임 종료 문구를 출력할 수 있다.
    ```
    게임이 종료되었습니다!
    ```
- [x] 승패를 원인과 함께 출력할 수 있다.
    ```
    초의 승리입니다.
    ```
