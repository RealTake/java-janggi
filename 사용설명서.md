# 사용 설명서

- 한나라는 아래 쪽 진영
- 초나라는 위 쪽 진영

1. 게임을 시작하면, 각 진영 별로 상, 마 배치를 선택할 수 있습니다.
2. 상, 마 배치를 선택하면, 게임이 시작됩니다.
3. 게임이 시작되면, 각 진영 별로 기물을 선택하여 움직일 수 있습니다.
4. 좌표를 입력해 어떤 기물을 어떤 위치로 움직일 지 선택합니다.
    - 좌표 입력 예시: `1,2`, `1,3`
5. 움직인 위치에 다른 기물이 있으면 해당 기물을 제거합니다.
6. 왕이 잡히면 게임이 종료됩니다.
